# sales-data-analysis
Analyzed sales performance across regions and product categories, identifying revenue trends and creating visualizations to support  data-driven business recommendations

## Tools Used
- Python
- Pandas (data manipulation)
- Matplotlib (visualization)

## Key Findings
- West region generated the highest revenue
- Technology products were the top-performing category
- Identified seasonal trends to optimize inventory planning

## Business Recommendations
- Focus marketing efforts on high-performing regions
- Improve strategies for underperforming categories
- Monitor monthly trends for promotional planning
